# get started with GitHub

Hello and welcome!


### What are we doing

Whilst many of us may have used tools such as Trello, JIRA and Azure DevOps to manage our work many have yet to see how to do this with GitHub. This course leads you through some simple activities to manage work in GitHub.

## Step 1: Set up the board

Create a Project board in your repository using the projects tab, selecting **Create a Project** then  name your board **Sprint1**
